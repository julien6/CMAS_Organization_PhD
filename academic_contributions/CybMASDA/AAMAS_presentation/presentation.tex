            Measuring how agents' behavior match a functional and structured organization?

            \vspace{0.8em}
      \item \textbf{How can we control organizational fit?} \\
            Guiding agents' training to align with predefined roles and goals?
    \end{enumerate}

  \end{block}

\section{Related works \& Contribution}

\begin{frame}{Related works \& Contribution}{Evaluation/Control in organizational fit?}

  \begin{columns}
    \begin{column}{0.4\textwidth}
      \centering
      \begin{block}{Limitations}
        \begin{itemize}
          \item No direct equivalent of the organizational fit concept.
          \item No systematic way to quantify how learned behaviors align with an implicit or given organization.
          \item No general framework for controlling/guiding agents in an organizational manner.
        \end{itemize}
      \end{block}
      \begin{exampleblock}{Contribution}
        Organizational model + MARL? \\ \ \ \ \ $\rightarrow$ \textbf{MOISE+MARL}
      \end{exampleblock}
    \end{column}
    \hspace{0.cm}
    \begin{column}{0.6\textwidth}
      \centering
      \begin{footnotesize}
        "Control of organizational fit"?
        \begin{itemize}
          \item Wilson et al.~\autocite{wilson2008learning}: Role transfer across environments in Multi-Agent MDPs, but lacks abstraction.
          \item Berenji and Vengerov~\autocite{berenji2000learning}: Coordination and role inference in UAVs — task-specific, not generalizable.
          \item Yusuf and Baber~\autocite{yusuf2020inferential}: Bayesian inference for task coordination — lacks organizational alignment metrics.
          \item Serrino et al.~\autocite{serrino2019finding}: Social role inference from interaction — limited to short-term operational roles.
        \end{itemize}

        \vspace{0.5cm}

        "Evaluation of organizational fit"?
        \begin{itemize}
          \item CPO~\autocite{achiam2017cpo}: Uses safety constraints to regularize learning — no extension to organizational constraints.
          \item Ray et al.~\autocite{ray2019benchmarking}: Lagrangian reward shaping — no dynamic action spaces constraining.
          \item Safe exploration~\autocite{garcia2015comprehensive,alshiekh2018safe}: Focus on avoiding unsafe states — not on role compliance.
          \item HRL~\autocite{ghavamzadeh2006hrl}: Task decomposition aligns well with hierarchies — but internally enforced.
        \end{itemize}
      \end{footnotesize}
    \end{column}
  \end{columns}

\end{frame}

    \begin{column}{0.7\textwidth}
      \vspace{-2ex}

      \begin{center}
        \begin{minipage}{0.95\linewidth}
          \centering
          \begin{block}{Markovian models for MARL: Dec-POMDP}
            {\small
              Decentralized Partially Observable Markov Decision Process (Dec-POMDP)~\autocite{Oliehoek2016}
              \begin{itemize}
                \item considers multiple agents in a similar MAS fashion
                \item stochastic processes for uncertainty in environmental changes including observations;
                \item reward function is common to agents which fosters training for collaborative oriented actions~\autocite{Beynier2013}
              \end{itemize}
            }
            \

            { \scriptsize

            $(S,\{A_i\},T,R,\{\Omega_i\},O,\gamma)$ , where
            \begin{itemize}
              \item $S = \{s_1, ..s_{|S|}\}$: The set of the possible states;
              \item $A_{i} = \{a_{1}^{i},..,a_{|A_{i}|}^{i}\}$: The set of the possible actions for agent $i$;
              \item $T$ so that $T(s,a,s') = \probP{(s'|s,a)}$ : The set of conditional transition probabilities;
              \item $R: S \times A \times S \rightarrow \mathbb{R}$: The reward function
              \item $\Omega_{i} = \{o_{1}^{i},..,o_{|\Omega_{i}|}^{i}\}$: The set of observations for agent $ag_i$;
              \item $O$ so that $O(s',a,o) = \probP{(o|s',a)}$ : The set of conditional observation probabilities;
              \item $\gamma \in [0,1]$, the discount factor.
            \end{itemize}

            }

          \end{block}

        \end{minipage}
      \end{center}

    \end{column}

  \end{columns}

\end{frame}

\section{Experimental Setup}

\begin{frame}{Experimental setup}{Evaluation metrics \& protocol}

  \begin{block}{Goals of the experimental framework}
    \begin{itemize}
      \item Assess whether organizational constraints improve:
            \begin{itemize}
              \item \textbf{Behavioral alignment} with structured roles and goals.
              \item \textbf{Learning stability and convergence}.
              \item \textbf{Explainability and robustness} of trained agents.
            \end{itemize}
      \item Validate that the proposed organizational specifications are faithfully reflected in agent behavior.
    \end{itemize}
  \end{block}

  \vspace{0.5em}

  \begin{block}{Evaluation metrics}
    Cumulative Reward, Reward Standard Deviation, Convergence Rate, Constraint Violation Rate, Consistency Score, Robustness Score, Organizational Fit Level
  \end{block}

  \vspace{0.5em}

  \begin{block}{Evaluation protocol}
    \begin{itemize}
      \item \textbf{Reference Baseline (RB)}: MARL training without organizational constraints.
      \item \textbf{Organizational Baseline (OB)}: Training with predefined MOISE+ specifications.
      \item Metrics are computed and compared across environments.
    \end{itemize}
  \end{block}

\end{frame}

\begin{frame}{Results}{Organizational Fit, Performance, and Comparison}

  \begin{itemize}

    \item \textbf{Performance \& Stability}:
          \begin{itemize}
            \item Reward STD ↓ (Overcooked-AI \& MAPPO: 15.6 → 10.4, Predator-Prey \& MADDPG: 21.5 → 15.2)
            \item Convergence ↑ (CybORG \& COMA: 0.70 → 0.86, Warehouse \& Q-Mix: 0.74 → 0.88)
            \item Robustness ↑ (Overcooked-AI \& MAPPO: 0.71 → 0.89) (\textit{provided bias})
            \item Zero constraint violations at high hardness
          \end{itemize}

    \item \textbf{Versus AGR+MARL} (no "goals"):
          \begin{itemize}
            \item Org. Fit: +33\% (Overcooked-AI), Robustness: +0.14 (Predator-Prey)
            \item Reward ↑ (Warehouse: 307.1 vs. 278.6)
            \item[] $\rightarrow$ \textbf{Gains from intermediary goals}
          \end{itemize}

    \item \textbf{Overall Consistency} $\geq$ 0.76 \quad $\rightarrow$ \textbf{Behaviors} match roles/goals (+ manual inspection)
    
    \item \textit{Preliminary results: Training time seems to scale \textbf{linearly}-like within a small number of constraints}

  \end{itemize}

\end{frame}
