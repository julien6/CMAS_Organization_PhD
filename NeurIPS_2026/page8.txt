Conclusion

World models for multi-agent systems should distinguish what is rule-covered from what is residual.
NS-MAWM formalizes this decomposition and provides a unified interface for symbolic regularization, projection, and residual dynamics. Under correct rules and masks, the hard variants guarantee
zero covered-feature violations, while residual dynamics changes the neural learning target.
Partial symbolic coverage improves long-horizon rollout fidelity, reduces semantic violations, and
increases downstream planning return under matched budgets. The main diagnostic lesson is that
zero post-correction validity is not enough: it must be paired with low pre-correction violations or
small correction magnitudes to indicate a useful model.

255

Future work includes automatic rule acquisition, scalable symbolic coverage representations, and
more efficient implementations for larger agent teams.
